# 🖼️ Image Gallery Web App

A responsive and interactive **Image Gallery** built using **HTML, CSS, and JavaScript**.
It includes a modern layout, lightbox preview, navigation controls, filters, and smooth UI effects.

---

## 🚀 Features

* 📱 **Responsive Design** (works on mobile, tablet, desktop)
* 🖼️ **Grid-based Gallery Layout**
* 🔍 **Lightbox View** for enlarged images
* ⬅️➡️ **Next/Previous Navigation**
* 🎯 **Category Filters** (e.g., Nature, City)
* ✨ **Hover Effects & Smooth Transitions**
* ⌨️ *(Optional)* Keyboard Navigation support

---

## 🛠️ Technologies Used

* **HTML5** – Structure
* **CSS3** – Styling & responsiveness
* **JavaScript (Vanilla)** – Functionality

---

image-gallery/

---

## ⚙️ How to Run

1. Download or clone the repository
2. Open the project folder
3. Double-click on `index.html`

Or open it in your browser:

```
Right click → Open with → Browser
```

---

## 🎮 How to Use

* Click on any image to open it in **lightbox mode**
* Use **Next (❯)** and **Previous (❮)** buttons to navigate
* Click **×** to close the lightbox
* Use filter buttons to display specific categories

---

## 🎨 Customization

You can easily:

* Add more images inside the `.gallery` section
* Create new categories by adding class names
* Modify colors and styles in `style.css`
* Enhance functionality in `script.js`
